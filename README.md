# Fibonacci_Sequence_In_JavaScript
## The Fibonacci sequence is the integer sequence where the first two terms are 0 and 1. After that, the next term is defined as the sum of the previous two terms.

0, 1, 1, 2, 3, 5, 8, 13, 21, ...
# 1+1=2                         
# 1+2=3                        
# 2+3=5                         
# 3+5=8                         
# 5+8=13
